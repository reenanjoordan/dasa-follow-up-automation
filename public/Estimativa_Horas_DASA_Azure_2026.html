08 Premissas e Consideracoes

Premissas da Estimativa

  • Equipe de 3 profissionais (2 Devs Senior + 1 GP/Tech Lead) dedicados 100%
  • DASA provisionara acessos Azure, ServiceNow e Teams dentro de 2 semanas
  • Stack 100% Azure-DASA: Azure Logic Apps + Azure Functions + Azure SQL + Azure OpenAI GPT-4o
  • Sprints de 2 semanas com entregas incrementais (metodologia Scrum)
  • 806h de esforco de desenvolvimento (403h por desenvolvedor)
  • Azure OpenAI GPT-4o para categorizacao com meta de 92% acuracia
  • Integracoes: ServiceNow TSM API, Microsoft Teams Graph API, Twilio API
  • Dashboard operacional React + Power BI embarcado
  • Sustentacao otimizada: R$ 7.500/mes (vs R$ 21.400 modelo original)
